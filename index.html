hypertext markup language (HTML) IS THE STANDARD MARKUPlanguage for creating web page and web applocations. HTML ( Hypertext Markup Language )은 웹 브라우저에 표시되도록 설계된 문서 의 표준 마크 업 언어 입니다 . CSS ( Cascading Style Sheets )와 같은 기술과 JavaScript 와 같은 스크립팅 언어 로 도움을받을 수 있습니다 .

웹 브라우저 는 웹 서버 또는 로컬 저장소 에서 HTML 문서를 수신 하여 문서를 멀티미디어 웹 페이지로 렌더링 합니다. HTML은 의미 적 으로 웹 페이지의 구조를 설명 하고 원래 문서의 모양에 대한 단서를 포함합니다.

HTML 요소 는 HTML 페이지의 구성 요소 입니다. HTML 구성을 사용하면 이미지 및 대화 형 양식 과 같은 기타 객체 가 렌더링 된 페이지에 임베드 될 수 있습니다. HTML은 제목, 단락, 목록, 링크 , 따옴표 및 기타 항목과 같은 텍스트의 구조적 의미 를 표시 하여 구조화 된 문서 를 작성하는 수단을 제공 합니다. HTML 요소는 꺾쇠 괄호를 사용하여 작성된 태그 로 구분 됩니다 . 같은 태그 와 직접 페이지에 내용을 소개합니다. 같은 다른 태그

서라운드 및 문서 텍스트에 대한 정보를 제공하며 다른 태그를 하위 요소로 포함 할 수 있습니다. 브라우저는 HTML 태그를 표시하지 않지만이를 사용하여 페이지 내용을 해석합니다.

HTML은 JavaScript 와 같은 스크립팅 언어로 작성된 프로그램을 포함 할 수 있으며 이는 웹 페이지의 동작과 내용에 영향을줍니다. CSS를 포함하면 내용의 모양과 레이아웃이 정의됩니다. 월드 와이드 웹 컨소시엄 (W3C)는 HTML과 CSS 표준의 현재 메인테이너 전 메인테이너는 1997 년부터 명시 적 표상 HTML을 통해 CSS의 사용을 권장하고있다 [2]
